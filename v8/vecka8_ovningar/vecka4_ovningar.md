# Vecka 4: Context API och global state

## Övning 1: ThemeContext
- Skapa en ThemeContext med Provider och custom hook.
- Byt tema (light/dark) och spara i localStorage.

## Övning 2: AuthContext
- Skapa en AuthContext med reducer för status, user, token, error.
- Bygg login-formulär och skyddad vy (AccountPanel).

## Övning 3: Kombinera Providers
- Skapa AppProviders som wrappas runt App.

## Övning 4: Konsumera context
- Använd useTheme och useAuthState/useAuthActions i olika komponenter.

## Övning 5: Reflektion
- Diskutera när det är lämpligt att använda Context och när det är bättre med lokalt state.
